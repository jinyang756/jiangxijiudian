江西酒店管理面板 Supabase 部署步骤

==========================================

1. 登录 Supabase 账户
   - 打开终端
   - 运行命令: npx supabase login
   - 按提示在浏览器中完成登录

2. 导航到管理面板目录
   - 运行命令: cd admin-panel

3. 执行部署命令
   - 运行命令: npx supabase deploy
   - 如果提示选择项目，请选择您的江西酒店项目

4. 配置环境变量（重要）
   - 登录 Supabase Dashboard: https://app.supabase.com
   - 选择您的江西酒店项目
   - 进入 Settings -> Configuration
   - 在 Environment Variables 部分添加以下变量:
     
     名称: VITE_APP_DB_URL
     值: YOUR_SUPABASE_URL
     
     名称: VITE_APP_DB_POSTGRES_PASSWORD
     值: YOUR_SUPABASE_ANON_KEY

5. 访问部署的管理面板
   部署完成后，通过以下 URL 访问:
   YOUR_SUPABASE_URL/projects/YOUR_PROJECT_ID/static/admin-panel

注意事项:
- 如果遇到权限问题，请确保您有足够的权限访问 Supabase 项目
- 如果部署失败，请检查网络连接和 Supabase CLI 是否正常工作
- 详细的操作指南请参考 DETAILED_DEPLOYMENT_INSTRUCTIONS.md 文件
