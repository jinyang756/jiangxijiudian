环境变量设置

请在下方输入您的 Supabase 环境变量以连接到数据库

获取凭据说明：

  1. 登录您的 Supabase 账户
  2. 进入您的项目
  3. 点击左侧菜单 "Project Settings" → "API"
  4. 复制 "Project URL" 和 "anon key"
Supabase URL不能为空
Supabase Anon Key不能为空

环境变量状态

点击"检查环境变量"按钮查看当前设置状态

连接测试结果

点击"测试连接"按钮测试数据库连接
